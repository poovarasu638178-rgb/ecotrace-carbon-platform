# EcoTrace 🌍
> Track it. Reduce it. Own it.

EcoTrace is a premium, enterprise-grade Carbon Footprint Awareness Platform. It combines custom calculators, real-time data visualisations, streak tracking milestone badge achievements, and a chat simulator to model and reduce your individual carbon footprints.

Designed with **Linear.app style dark themes**, **Stripe style canvas particle physics**, **Notion style typography (Inter + Space Grotesk)**, and **Vercel style interactive charts**.

---

## 🚀 Key Features

* **Multi-Step Footprint Calculator**: Granular parameter configuration tracking transport, diet, energy utility consumption, and consumption purchases.
* **EcoAI Simulator**: Ask questions or simulate reductions like "cut electricity by 30%" or "diet vegan" to see forecast changes. Integrates direct fetch requests to Anthropic's Claude API with fallback simulator checks.
* **Gamification & Ranks**: Evaluate 10 unlockable milestone badges (e.g. *Clean Commuter*, *Zero Waste Champion*) based on calculation history.
* **Interactive Data Visualization**: Embedded Chart.js doughnut breakdowns and historical emission line charts with high-performance 2D Canvas fallbacks.
* **130 Automated Diagnostics**: Built-in test suite evaluating calculations, range parameters, security sanitization, and state persistence.

---

## 🛠️ Tech Stack

* **Structure**: HTML5
* **Logic Engine**: Vanilla JavaScript (MVC Model-View-Controller framework, Observer patterns, and Factory logic)
* **Styling**: Vanilla CSS3 Custom Variables & Tokens
* **Libraries & Assets**: 
  * Google Fonts (Inter & Space Grotesk)
  * DOMPurify (Sanitization)
  * Chart.js (Data Vis)
  * Animate.css (Micro-animations)

---

## 💻 How to Run Locally

Since EcoTrace is a zero-build-step application, you can load and run it directly in your browser:

1. Clone or download this project.
2. Open `index.html` in any web browser.
3. To view or run the diagnostic test suite drawer, toggle the **Automated Test Runner** overlay in the bottom right corner of the interface.
